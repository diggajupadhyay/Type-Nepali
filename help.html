How to use Nepali Typing Extension

This extension allows you to type in Nepali on any website.

Features:

  • Real-time transliteration from English to Nepali.
  • Enable/disable the extension with a single click.
  • Customizable settings.

How to use:

  1. Click on the extension icon to open the popup.
  2. Click the "On" button to enable the extension.
  3. Start typing in English, and it will be automatically converted to Nepali.
